# London ANPR Dashboard

Real-time traffic monitoring system with ANPR cameras, live map, alerts, and admin panel.

## Features
- Live Leaflet map with 100+ cameras
- Real-time alerts with sound & browser notification
- ShadCN + Tailwind UI (dark mode)
- Firebase Auth + Firestore
- Admin panel (CRUD cameras)
- Export reports (CSV/PDF)

## Tech Stack
- Next.js 16 (App Router)
- React 19
- Firebase (Auth, Firestore, Hosting)
- Leaflet + Custom Markers
- ShadCN/UI + Tailwind CSS
- TypeScript
